Spectrometer utility tool.

Introduction

Aim of this website is to deliver online facility for processing of spectrometer data (mainly from ASD FieldSpec 3 and 4).

Processed data consist of averaged measurements for particular measurement point, standard deviation of measurements and number of vegetation indices calculated on averaged spectra.

Example of processed file data structure:
Wavelength; Meas1.001; Meas1.002; Meas1.003; Meas1.004; Average; Std dev.; Meas2.001;
350 ; 0.2153 ; 0.2115 ; 0.2233 ; 0.2513 ; 0.22535 ; 0.01557 ; ... ;
351 ; 0.2153 ; 0.2211 ; 0.2107 ; 0.2121 ; ... ; ... ; ... ;
352 ; 0.1892 ; 0.1920 ; 0.1944 ; 0.2012 ; ... ; ... ; ... ;
Lits of calculated vegetation indices:
ARI1 ARVI CAI CRI1 CRI2 EVI
MSI NDII NDLI NDVI705 NDWI OSAVI
PRI PSRI SIPI SR TCARI VARI
VOG1 VOG2 VOG3 WBI MCARI MSAVI
MSR NDVI RDVI SAVI TVI

Instruction of usage

Before you process data you have to prepare them according to following advice.

Most important issue is naming of measurement points. In order to properly group measurements made at the same measurement point you have to name all measurements following bellow pattern: PointName.measurement_number example: Spruce.000

IMPORTANT -> PointName and measurement_number have to be separated with . (dot). Failing to do so will make proper grouping impossible.

Fields in your file have to be separated using ; (semicolon). This is easiest to archive if you save your file as .csv using LibreOffice or similar program (remember to specify separation symbol as ;).
